## 🛠️ Tecnologías Utilizadas

El proyecto fue construido utilizando tecnologías web estándar (Vanilla de desarrollo frontend) 

*   **HTML5:** Estructuración semántica del contenido de la página para mejorar la accesibilidad y el SEO.
*   **CSS3:** Diseño visual adaptativo (Responsive Design) utilizando Flexbox/Grid y Media Queries para asegurar la compatibilidad con dispositivos móviles.
*   **JavaScript (ES6+):** Lógica e interactividad del lado del cliente (manipulación del DOM, eventos y validación).
*   **Recursos Multimedia:** Optimización de imágenes para una carga rápida en el navegador.

---

## 📂 Estructura del Repositorio

Para mantener las buenas prácticas del desarrollo, el código está organizado de la siguiente manera:

*   `index.html` - Archivo principal y punto de entrada del sitio web.
*   `css/` - Carpeta que contiene las hojas de estilo (`styles.css`).
*   `js/` - Carpeta destinada a los scripts de interactividad (`script.js`).
*   `img/` - Carpeta donde se almacenan las imágenes, iconos y logotipos.
*   `README.md` - Documentación del proyecto.
